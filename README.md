# S3TP

A new implementation of a simple stupid satellite transport protocol.

To be continued ...

## Build instructions

### Linux

Install the following dependencies:

* gcc or clang
* cmake
* make

```
mkdir build
cd build
cmake ..
make
```

### Rodos

Install the following dependencies:

* gcc or clang
* meson
* ninja

```
meson build
ninja -Cbuild
```
